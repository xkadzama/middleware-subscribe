
#### Клонирование репозитория
```bash
git clone https://github.com/xkadzama/middleware-subscribe.git
cd middleware-subscribe
```

#### Установка зависимостей
```bash
pip install -r requirements.txt
```

#### Установить свой токен
```bash
В файле .env
API=ТВОЙ_ТОКЕН
```

#### Запуск бота
```bash
python bot.py
```
